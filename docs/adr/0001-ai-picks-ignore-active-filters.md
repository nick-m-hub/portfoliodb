# AI Picks ignore active database filters

On the database page, the AI Recommendation feature surfaces 3 portfolios ("AI Picks") pinned above the filtered grid. These picks are not constrained by whatever category, risk, asset exposure, or strategy filters the user has active — they always reflect the AI's answer to the user's goal description against the full portfolio set.

The alternative — constraining picks to the filtered set — was rejected because a user with narrow filters active could get zero visible AI results, which is a worse outcome than seeing a pick flagged as "outside your filters." The filter-bypass is intentional; any pick that falls outside the active filters is labelled inline on its card.
