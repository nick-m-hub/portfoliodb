# Lump Sum vs. DCA measures both strategies over the same total window

Both the lump sum and DCA scenarios run from the same start date to the same end date. For a 10-year horizon with a 12-month DCA period, lump sum is fully invested for 120 months and DCA deploys $1/N per month for the first 12 months of those same 120 months. The DCA period is contained within the Investment Horizon, not added on top of it.

## Considered Options

The alternative ("post-DCA" framing) measures both strategies for the same duration *after* the DCA period completes — meaning DCA runs 11 years total vs. lump sum's 10. This was rejected because it answers a different question: "what if I had already finished DCA?" rather than "I have money today — what do I do with it?" The Vanguard study most users will have read uses the same-total-window framing, so our results are directly comparable to the commonly cited finding that lump sum wins ~2/3 of the time.
